## рЯЪА Quick Start (5 minutes)

### Step 1 вАФ Use this template

Click the green **"Use this template"** button at the top of this page, then choose **"Create a new repository"**. Give it a name like `my-stats-wales-dashboard`.

### Step 2 вАФ Open in Codespaces

In your new repository, click the green **"Code"** button вЖТ **"Codespaces"** tab вЖТ **"Create codespace on main"**.

A full VS Code environment will open in your browser. No installs. No admin rights needed. вШБпЄП

### Step 3 вАФ Run the notebook

Open `workshop/fetch_data.ipynb` and run each cell from top to bottom. The notebook shows you all the available Stats Wales datasets вАФ **pick one that interests you**, paste its ID, and the notebook does the rest (fetching data, creating a chart, exporting for the dashboard). Then practise committing your changes with Git.

Open [`WORKSHOP_GUIDE.md`](./WORKSHOP_GUIDE.md) for the full walkthrough.
